# Nama Proyek dan Deskrips singkat 
**web penjualan bisnis UMKM Marshmallow** 
Sebuah aplikasi web sederhana untuk membantu UMKM yang menjual Marshmallow

## Fitur utama 
* Menambah data Marshmallow baru
* Hapus data Marshmallow yang tidak diperlukan
* Mengubah data Marshmallow sesuai kebutuhan
* Tampilan responsif untuk desktop dan mobile

# Apa Fungsi dari Pembuatan web ini Untuk Kami?
Fungsinya ialah unutk memenuhi tugas kami sebagai anggota UKM Programming, yakni membuat pemograman CRUD (Create, Read, Update, Delete) 

# Bagaimana Cara Mengakses web kami?
[Tautan Tugas Akhir kelompok Pemula](https://workmarshmallow.netlify.app)
Link ini dapat diakses dari  desktop maupun mobile

# Bahasa Pemmograman Apa Saja yang Digunakan Untuk pembuatan Web ini?
HTML        :Sebagai Struktur Dasar
CSS         :Sebagai penghias
JavaScript  :Untuk menambah interaksivitas antar pengguna dan web
